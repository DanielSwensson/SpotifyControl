# SpotifyOsxController

CLI for Spotify running on OSX

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'spotify_osx_controller'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install spotify_osx_controller
